Garanti BBVA Virtual POS Laravel Paketi

Gelişmiş, esnek ve kullanımı kolay Garanti Sanal POS entegrasyonu.

1. Kurulum

Paketi Composer aracılığıyla projenize dahil edin:

composer require developertugrul/garanti-pos

Laravel 5.5+ sürümlerinde Package Auto-Discovery sayesinde servis sağlayıcı otomatik olarak kaydedilir. Daha eski bir sürüm kullanıyorsanız config/app.php dosyanıza ekleyin:

'providers' => [
    // ...
    Developertugrul\GarantiPos\GarantiPosServiceProvider::class,
],

'aliases' => [
    // ...
    'GarantiPos' => Developertugrul\GarantiPos\Facades\GarantiPos::class,
],

2. Yapılandırma

Ayar dosyasını dışa aktarın:

php artisan vendor:publish --provider="Developertugrul\GarantiPos\GarantiPosServiceProvider"

.env dosyanıza bankadan aldığınız bilgileri ekleyin:

GARANTI_POS_MODE=PROD # veya TEST
GARANTI_POS_TERMINAL_ID=12345678
GARANTI_POS_PROV_USER_ID=PROVAUT
GARANTI_POS_PROV_PASSWORD=Sifre123
GARANTI_POS_MERCHANT_ID=1234567
GARANTI_POS_STORE_KEY=3DSecureAnahtari
GARANTI_POS_CURRENCY=949 # TL: 949, USD: 840, EUR: 978

3. Normal Satış (Non-3D)

Dikkat: 3D'siz (Non-3D) işlemler banka tarafından özel olarak yetkilendirilmelidir. 
use Developertugrul\GarantiPos\Facades\GarantiPos;

$orderData = [
    'order_id' => 'Siparis123',
    'amount' => '100', // 1.00 TL için 100 kuruş gönderilir
    'installment' => '', // Taksit yoksa boş bırakın, varsa '3', '6' vb.
    'ip_address' => request()->ip(),
    'email' => 'musteri@example.com',
    'description' => 'Sipariş Açıklaması'
];

$cardData = [
    'number' => '5400111122223333',
    'expire_month' => '12',
    'expire_year' => '25', // Son 2 hane
    'cvv' => '123'
];

$response = GarantiPos::pay($orderData, $cardData);

if ($response['Transaction']['Response']['ReasonCode'] == '00') {
    echo "Ödeme Başarılı!";
} else {
    echo "Hata: " . $response['Transaction']['Response']['ErrorMsg'];
}

4. 3D Secure Satış (3D Pay)

3D Secure işlemlerinde kart sahibini bankanın ortak ödeme veya şifre girme sayfasına yönlendirmeniz gerekir.

$orderData = [
    'order_id' => 'Siparis123',
    'amount' => '100', // Kuruş cinsinden
    'installment' => '',
];

$cardData = [
    'number' => '5400111122223333',
    'expire_month' => '12',
    'expire_year' => '25',
    'cvv' => '123'
];

$successUrl = route('payment.success');
$errorUrl = route('payment.error');

// Otomatik submit olan HTML formu döner. 
// Bunu doğrudan Blade dosyanızda {!! $formHtml !!} şeklinde basabilirsiniz.
$formHtml = GarantiPos::build3DForm($orderData, $cardData, $successUrl, $errorUrl, 'sales');

return view('payment.redirect', compact('formHtml'));

5. İptal İşlemi (Cancel)

İptal işlemi sadece gün sonu alınmamış (aynı gün içindeki) işlemler için yapılabilir.

$orderId = 'Siparis123';
$response = GarantiPos::cancel($orderId);

if ($response['Transaction']['Response']['ReasonCode'] == '00') {
    echo "İptal Başarılı!";
}

6. İade İşlemi (Refund)

Gün sonu alınmış önceki günlere ait işlemlerin tamamını veya bir kısmını iade etmek için kullanılır.

$orderId = 'Siparis123';
$refundAmount = '50'; // 0.50 TL iade
$response = GarantiPos::refund($orderId, $refundAmount);
